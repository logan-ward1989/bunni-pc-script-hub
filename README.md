## Overview

Bunni PC is a Windows Roblox script executor built around Lua injection and a straightforward script workflow. It is aimed at users who want a desktop utility with quick script access, a simple layout, and execution steps that stay minimal.

The project also includes a built-in script hub with 500+ scripts, persistent queue storage, batch execution mode, and an auto-update engine. With compatibility for Windows 10 and 11, it fits into the typical Roblox scripting workflow while keeping the interface lightweight and easy to work with.
